# **App Name**: Bovindex

## Core Features:

- Bovine Vision Classifier: Upload or capture photos to identify specific cattle and buffalo breeds using an advanced computer vision tool to recognize physical characteristics.
- Trait Profiler: Automatically generates detailed data on the identified breed, including origin, milk yield estimates, and environmental adaptability using a knowledge retrieval tool.
- Dynamic Dashboard: A professional-grade interface to manage recent scans and toggle between classification modes.
- Local Scan Ledger: Maintains a history of previously identified animals in browser storage without the need for an external database.
- Interactive Highlight Maps: Overlays visual markers on the image to show which features of the animal (horns, coat pattern, dewlap) led to the classification result.
- Care & Management Guide: Generates breed-specific nutrition and health management tips tailored to the animal's life stage detected in the photo.

## Style Guidelines:

- Primary color: Earth Sienna (#7A4D1F) inspired by rich organic soil and bovine coats.
- Background color: Warm Bone (#FAF8F5) providing a soft, accessible canvas for agricultural imagery.
- Accent color: Terracotta Red (#BD2E2E) used for high-impact call-to-action buttons and critical data highlights.
- Headline font: 'Alegreya' (serif) for an elegant, established, and authoritative agricultural presence. Body font: 'Inter' (sans-serif) for high readability in dense data tables.
- Thin-stroke minimalist icons that focus on livestock anatomy and rural farming utility.
- A spacious, editorial-style layout utilizing a split-pane view to balance the image upload process with immediate result summaries.
- Soft fade-in transitions for breed profile components and a subtle pulsing 'scan line' overlay during AI processing.
